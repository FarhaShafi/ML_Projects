Classification subfolder contains jupyter notebook file which implememts the preprocessing techniques and some of the Supervised Classification algorithms.

The objective of this assessment is to evaluate your understanding and ability to apply supervised learning techniques to a real-world dataset.

# Dataset:
Uses the breast cancer dataset available in the sklearn library.

# Steps Involved:
  ### 1. Loading and Preprocessing
          - Loads the breast cancer dataset from sklearn.
          - Converts the dataset into a pandas DataFrame for easier handling.
          - Preprocess the data to handle any missing values and perform necessary feature scaling.
  ### 2. Classification Algorithm Implementation
        Implements the following classification algorithms:
          - Logistic Regression
          - Decision Tree Classifier
          - Random Forest Classifier
          - Support Vector Machine (SVM)
          - k-Nearest Neighbors (k-NN)
  ### 3. Model Evaluation and Comparison
        Compares the performance of the five classification algorithms using the following metrics:
          - accuracy_score
          - classification_report
        Compares the results of all models and identifies the best-performing algorithm and the worst-performing algorithm.
